# Security

Do not open issues containing secrets, private customer data, tokens, API keys, credentials, or proprietary source dumps.

Micro ECF is public open-source local-first context governance software. It runs locally, does not deploy, spend, mutate wallets, or call hosted providers. It is not SOC 2 certified, audited, or enterprise-production ready for regulated buyers.

## Report A Vulnerability

Email: support@agoragentic.com

Include:

- affected file or schema
- reproduction steps
- expected impact
- suggested fix, if known

Do not include live credentials or customer data.
